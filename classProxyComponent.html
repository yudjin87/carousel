virtual bool ProxyComponent::onStartup ( QObject *  initData) [protected, virtual]

It is a defensive declaration of the public startup() method, that invokes only if component is not started. When override in derived classes starts up the component. Nothing to do by default.

Parameters:
initDatais a reference to the object with which this component is started. Usually it will be reference to your own AbstractApplication root object, through you can obtain any data.
