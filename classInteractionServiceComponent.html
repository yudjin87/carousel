Detailed Description

This class represents a component, that creates and registers interactive services in the service locator.

The InteractionServiceComponent is a main component of GUI based application due to the IInteractionService. If your components add new operations or menus, or dockable widgets - they should be a child to InteractionServiceComponent component. This mean a dependency of component parent name: "Interactivity"

Registered services:

  • IDialogService;
  • IInteractionService;
