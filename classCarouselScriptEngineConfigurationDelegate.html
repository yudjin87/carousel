Detailed Description

It is a default implementation of the IScriptEngineConfigurationDelegate interface.

Each time when new QScriptEngine is created to evaluate some JavaScript, this the configureFromComponent() method is invoked by the ScriptingService for each component to get it a chance to register custom types, new functions and other objects. Also it is called when new component is started at runtime.
